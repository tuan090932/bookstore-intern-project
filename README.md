# API ADMIN LARAVEL
### Environmental information
- PHP: 8.2.10
- MySQL: 8.0.24
- Laravel: v10.48.12 

### Setup

Start Laravel:

```
php artisan serve
```

Install composer

```
composer install
```

Copy .env

```
cp .env.example .env
```

Generate key

```
php artisan key:generate
```

Run migration

```
php artisan migrate
```

Run seeder

```
php artisan db:seed
```

Link:

```
http://localhost:8000
```
